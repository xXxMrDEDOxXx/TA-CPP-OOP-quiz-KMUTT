## คำอธิบาย
ให้นักศึกษาเขียนโปรแกรมภาษา C++ สำหรับตรวจสอบคำตอบของเกม ซูโดกุ (Sudoku) โดยใช้คุณสมบัติการสืบทอดคลาส (Inheritance) อย่างเหมาะสมเเละจะกำหนดโครงสร้างคลาสไว้และให้นักศึกษาเติมโค้ดในส่วนที่กำหนด
- พี่จะมีคลาสพื้นฐาน ```Puzzle``` สำหรับตรวจสอบ format ของซูโดกุ และคลาสลูก ```SudokuPuzzle``` สำหรับตรวจสอบคำตอบซูโดกุ
- คลาส ```Puzzle``` ทำหน้าที่ตรวจสอบเงื่อนไขทั่วไปของตารางตัวเลข เช่น ขนาดตารางและช่วงของตัวเลขที่ใช้ (1-9) ทั้งเเนวตั้งเเละเเนวนอน

```cpp
class Puzzle {
protected:
    std::vector<std::vector<int>> grid;

public:
    Puzzle(const std::vector<std::vector<int>>& initialGrid);
    virtual ~Puzzle() = default;
    virtual bool verifySolution();
};

```
- คลาส ```SudokuPuzzle``` สืบทอดจาก ```Puzzle``` เพื่อเพิ่มตรรกะการตรวจสอบตามกติกาของซูโดกุ
- การตรวจสอบคำตอบซูโดกุที่ถูกต้อง: ในแต่ละแถว แต่ละคอลัมน์ และแต่ละบล็อกย่อย 3×3 ของตารางขนาด 9×9 จะต้องมีตัวเลขตั้งแต่ 1 ถึง 9 อย่างละหนึ่งครั้งเท่านั้น (ไม่มีซ้ำ)
  
```cpp
class SudokuPuzzle : public Puzzle {
public:
    SudokuPuzzle(const vector<vector<int>>& initialGrid) 
        : Puzzle(initialGrid) {
    }

    bool verifySolution() override {
    }
};
```

### สิ่งที่นักศึกษาต้องทำ
**1.เติมโค้ดในเมธอด** ```Puzzle::verifySolution()``` เพื่อทำการตรวจสอบเบื้องต้นดังนี้:
- ตรวจสอบว่าตาราง ```grid``` เป็นขนาด 9 x 9 หากขนาดของแถวหรือจำนวนแถวไม่เท่ากับ 9 ให้คืนค่า ```false```
- ตรวจสอบว่าทุกช่องในตารางมีค่าตั้งแต่ 1 ถึง 9 เท่านั้น (ไม่มีเลข 0, เลขนอกช่วง หรือช่องว่างใด ๆ) หากพบค่าที่ไม่อยู่ในช่วงนี้ให้คืนค่า ```false```
- หากผ่านเงื่อนไขข้างต้นทั้งหมด ให้คืนค่า ```true``` เป็นการยืนยันว่าข้อมูลตารางเบื้องต้นอยู่ในรูปแบบที่ถูกต้อง

**2.เติมโค้ดในเมธอด** ```SudokuPuzzle::verifySolution()``` โดยมีขั้นตอนดังนี้:
- เรียกใช้การตรวจสอบเบื้องต้นจากเมธอด Puzzle::verifySolution() เพื่อยืนยันว่าตารางมีรูปแบบถูกต้องพื้นฐานก่อน (เช่น ขนาดถูกต้อง และมีแต่เลข 1-9) หากเมธอดของคลาสแม่คืนค่า false ให้เมธอดนี้คืนค่า false ทันที เนื่องจากข้อมูลตารางไม่ถูกต้องตามเกณฑ์พื้นฐาน
##### ถ้าเบื้องต้นผ่าน ให้ตรวจสอบกติกาซูโดกุเพิ่มเติม:
- ตรวจสอบแต่ละแถว: ว่าในแถวที่ i (0 ≤ i < 9) มีตัวเลข 1-9 ครบทุกตัวโดยไม่ซ้ำกัน
- ตรวจสอบแต่ละคอลัมน์: ว่าในคอลัมน์ที่ j (0 ≤ j < 9) มีตัวเลข 1-9 ครบทุกตัวโดยไม่ซ้ำกัน
- ตรวจสอบแต่ละบล็อกย่อย 3x3: แบ่งตารางออกเป็น 9 บล็อก (แต่ละบล็อกขนาด 3x3) ตรวจสอบแต่ละบล็อกว่ามีตัวเลข 1-9 ครบทุกตัวโดยไม่ซ้ำกันเช่นกัน
- return ค่าเป็นรูปเเบบ ```boolean``` หากเงื่อนไขถูกต้องทั้งหมดให้ return ค่าเป็น ```true``` หากไม่ใช่ return เป็น ```false```

3.**อย่าเขียนการแสดงผลภายในฟังก์ชันเหล่านี้** แต่ให้ใช้การคืนค่า (return) เพื่อส่งผลการตรวจสอบออกไปตามที่อธิบาย

```cpp

    {7, 9, 2, 1, 5, 4, 3, 8, 6},
```
